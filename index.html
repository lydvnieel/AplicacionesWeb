    Etiqueta <a>

    La etiqueta (a) es la mas importante de HTML, permite realizar referencias a otras paginas web
  • Tambien a documentos, imagenes, videos etc
  • Las referencias se suelen dominar dominios enlaces o vinculos
  • Tiene los siguientes esta etiqueta
  • href: Enlace de lo que se quiere enlazar e obligatorio
  • download: Descarga el recursoindicado de href en ves de abrirlo
  • target: objetivo donde se abre el enlace/recurso
  • ping: notifica al recurso que el usuario siguio el enlace
  • rel: tipo de enlace
  • type: Pista para el navegador sobre el recurso enlazado

    Imagenes etiqueta <img>

  • Src: Define la ruta de la imagen es obligatorio
  • alt: Es un texto que describe la imagen
  • width y height: Ancho y altura de la imagen
  • title: Texto que aparece cuando el cursor esta encima de la imagen
